# 환율알리미 v2.0.0 — Play 스토어 등록정보

## 앱 이름 (30자)
환율알리미 - 실시간 환율, 알림, 위젯

## 짧은 설명 (80자)
실시간 환율 시세와 목표 환율 도달 알림, 홈 화면 위젯까지. 완전히 새로워진 환율알리미

## 전체 설명 (4000자)
환율알리미가 완전히 새로워졌습니다. 더 빠르고, 더 정확하고, 더 보기 좋게 — 처음부터 다시 만들었습니다.

■ 실시간 환율 시세
· 달러(USD)·엔화(JPY)·유로(EUR)·위안(CNY) 등 58개 통화 실시간 고시
· 매매기준율과 함께 현찰 살 때/팔 때, 송금 보낼 때/받을 때 4종 가격까지 한눈에
· 최근 추이를 보여주는 미니 차트(스파크라인)와 등락 표시
· 관심 통화만 골라 나만의 환율 보드 구성

■ 목표 환율 알림 (완전 개편)
· 원하는 환율에 도달하면 푸시 알림으로 즉시 알려드립니다
· 이제 서버가 24시간 환율을 감시합니다 — 앱을 꺼두어도, 배터리 최적화와 무관하게 정확하게 도착
· 매매기준율뿐 아니라 현찰가·송금가 기준 알림도 지원
· 1회 알림 / 반복 알림 선택, ±0.5%·±1% 빠른 설정
· 평일 아침 환율 브리핑, 주요 통화 급변동 알림 (선택)

■ 환율 계산기
· 여러 통화를 한 화면에서 동시에 환산
· 소수점 금액 입력 지원
· 통화를 탭하면 기준 통화가 바로 전환
· 마지막 입력 금액 자동 저장

■ 상세 차트와 52주 범위
· 1주일부터 3년까지 기간별 환율 차트
· 최근 1년 최고가/최저가 대비 현재 위치를 한눈에
· 은행 우대율(수수료 할인율)을 적용한 실제 환전 가격 계산

■ 홈 화면 위젯
· 앱을 열지 않아도 홈 화면에서 바로 환율 확인
· 크기에 따라 1개 통화 또는 4개 통화 리스트 표시
· 라이트/다크 모드 자동 대응

■ 다크 모드
· 시스템 설정 연동 또는 수동 선택
